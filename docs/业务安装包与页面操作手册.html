精准营销自动化（业务版）安装与页面操作手册

版本日期：2026-03-30 · 面向业务执行同事 · 图文说明

一、安装包说明

  • 安装包文件：precision-marketing-auto-windows-oneclick.zip
  • 使用对象：业务执行同事（Windows）
  • 启动入口：scripts\\windows\\windows_start_ui.bat
建议先在测试账号验证 1 条任务

二、Windows 安装/更新

  1. 解压安装包到本地目录（例如 D:\\precision-marketing-auto-windows）。
  2. 双击运行：scripts\\windows\\windows_start_ui.bat。
  3. 浏览器打开：http://127.0.0.1:8790。
  4. 后续更新：用新 zip 覆盖原目录后，继续双击 windows_start_ui.bat。

四、标准操作流程（业务）

  1. 点击上传区域，上传 CSV/XLSX 任务文件。
  2. 系统自动拆分为任务列表多条记录。
  3. 如需补充素材：在任务行点击 人工修复。
  4. 点击 开始执行。
  5. 如有失败：点击 重试失败项 或任务行 重试。
  6. 在任务行点 日志 查看执行明细。

五、人工修复助手怎么用

  • 入口：任务列表 -> 人工修复
  • 可配置：小程序封面（按计划行）、朋友圈图片（按计划行）
  • 点击 保存修复配置 后，素材会回写该任务 CSV，重试即生效。

六、高级配置说明

  • 浏览器调试地址：默认 http://127.0.0.1:18800
  • 并发任务数：建议先 1
  • 保留浏览器(秒)：便于人工复核
  • 执行员工包含加盟区域：勾选后自动追加 xx加盟 节点
  • 失败通知 Webhook（飞书机器人）：失败自动推送结果摘要与日志尾部

七、失败协同排查（推荐）

  • 页面内看：任务行 日志
  • 本地看：ui_uploads/task_history
  • 飞书看：Webhook 通知中的错误摘要和日志尾部

八、常见问题

1) 页面打不开

  • 确认 windows_start_ui.bat 是否运行中
  • 检查端口占用：8790

2) CDP 连接失败

  • 错误：connect ECONNREFUSED 127.0.0.1:18800
  • 处理：先启动 Chrome 调试端口，再执行任务

3) 任务显示成功但后台没记录

  • 先看日志是否出现保存成功提示
  • 检查是否有页面校验拦截（结束时间、分配方式、素材必填）

4) 图片上传失败

  • 检查格式：jpg/png
  • 检查大小：<10MB
  • 检查分辨率是否符合业务系统限制
