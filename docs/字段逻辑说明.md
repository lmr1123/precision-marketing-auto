# 精准营销自动化字段逻辑说明（业务版）

版本日期：2026-03-30

## 1. 上传文件总体规则
- 任务上传支持 `CSV/XLSX`。
- 一行 = 一个计划任务（会拆成任务列表多条记录）。
- 发送渠道由文件字段 `发送渠道` 控制（支持中文名、多值、历史兼容）。
- 非社群渠道：`创建链接` 必填；社群渠道未填时默认使用系统社群创建链接。

## 2. 核心字段映射
| 业务字段（中文） | 内部字段 | 说明 |
|---|---|---|
| 计划名称 | `name` | 第1步计划名称 |
| 计划区域 | `region` | 第1步计划区域下拉 |
| 营销主题 | `theme` | 支持多值（用 `、` / `,` / `|` 分隔） |
| 场景类型 | `scene_type` | 第1步场景类型 |
| 计划类型 | `plan_type` | 第1步计划类型 |
| 计划开始时间 | `start_time` | 第1步计划时间开始 |
| 计划结束时间 | `end_time` | 第1步计划时间结束 |
| 发送时间 | `send_time` | 第1步发送时间 |
| 全局触达限制 | `global_limit` | 第1步触达限制 |
| 创建链接 | `create_url` | 打开页面入口（非社群必填） |
| 发送渠道 | `channels` | 渠道匹配逻辑核心字段 |
| 推送内容 | `push_content` | 按渠道自动映射到短信内容/发送内容 |
| 员工任务结束时间 | `step3_end_time` | 第3步结束时间（会员通消息/朋友圈/社群） |
| 社群任务分配方式 | `distribution_mode` | 仅社群渠道生效 |
| 执行员工 | `executor_employees` | 第3步执行员工选择 |
| 主消费营运区 | `main_operating_area` | 第2步区域选择 |
| 已领或已使用券规则ID | `coupon_ids_sheet_ref` | 第2步券规则ID来源（可引用sheet） |
| 购买目标商品编码 | `purchase_target_product_code` | 第2步商品编码来源（可引用sheet） |

## 3. 渠道与字段执行逻辑

### 3.1 短信
- 必填：计划基础字段、推送内容。
- 第2步：按数据决定是否执行（当前策略非社群一般执行）。

### 3.2 会员通-发客户消息
- 必填：员工任务结束时间、执行员工、推送内容。
- 可选：小程序卡片（名称/标题/链接/封面）。
- 可选：上传门店（指定门店分配）。

### 3.3 会员通-发客户朋友圈
- 必填：员工任务结束时间、执行员工、推送内容。
- 可选：朋友圈图片上传（最多9张，jpg/png，<10MB）。

### 3.4 会员通-发送社群
- 默认创建链接：`https://precision.dslyy.com/admin#/marketingPlan/addcommunityPlan?checkType=add`
- 必填：员工任务结束时间、推送内容、下发群名（若页面要求）。
- 分配方式：
  - 按条件筛选客户群：点击“选择员工”弹窗选择区域 -> 添加全部 -> 确定。
  - 导入门店：点击“上传门店”导入门店编码文件。
- 可选：小程序、图片（与会员通其他渠道逻辑一致）。

## 4. 书名号引用 sheet 规则（严格）
- `执行员工` 可填：`《目标门店1》`
  - 系统将读取同名 sheet，提取门店编码并生成上传文件。
- `购买目标商品编码` 可填：`《目标商品1》`
  - 系统读取同名 sheet 商品编码。
- `已领或已使用券规则ID` 可填：`《券规则ID1》`
  - 系统读取同名 sheet 券规则ID，并填到第2步所有券规则ID输入位。
- 必须包含书名号 `《》`；找不到 sheet 时：阻断报错。

## 5. 失败任务人工修复助手
- 任务列表点击 `人工修复`，可按计划行补充素材。
- 保存后回写任务CSV，再点 `重试` 即按最新素材重跑。

## 6. 失败日志与协同排查
- 本地持久化目录：`ui_uploads/task_history`
  - `任务ID.json`：任务快照+日志
  - `任务ID.log`：纯文本日志
  - `failed_tasks.ndjson`：失败索引
- 可配置飞书机器人 Webhook：失败自动推送摘要与日志尾部，便于跨区域协同。

## 7. 常见校验与阻断
- 非社群渠道缺创建链接：阻断。
- 社群渠道结束时间为空：阻断。
- 小程序封面格式不合法：阻断。
- 朋友圈图片尺寸/格式不合法：阻断。
- 书名号引用 sheet 不存在：阻断。
