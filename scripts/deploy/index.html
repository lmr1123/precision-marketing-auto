创建工具快速开始

  1. 下载 Windows 版安装包，右键“全部解压”。
  2. 进入解压后的文件夹，双击根目录 start.bat。
  3. 首次启动会自动准备运行环境，稍等 1-3 分钟。
  4. 自动打开 Chrome 和操作页 http://127.0.0.1:8790/simple 后即可使用。

Chrome 复核插件安装

  1. 点击上方“下载 Chrome 复核插件”。
  2. 右键 zip 文件，选择“全部解压”。
  3. 在 Chrome 地址栏输入 chrome://extensions/。
  4. 打开右上角“开发者模式”。
  5. 点击“加载已解压的扩展程序”。
  6. 选择解压后、里面直接有 manifest.json 的文件夹。

复核插件怎么用

  1. 打开业务系统里的计划编辑页或复核页。
  2. 点击 Chrome 右上角“精准营销复核助手”。
  3. 粘贴原始文本计划，或粘贴创建结果里的“复制复核数据”。
  4. 点击 文本复核。
  5. 如果有待复核字段，打开对应步骤/弹窗后再点 文本复核。
  6. 仍读不到时，再点 截图复核。
插件只是辅助复核，不影响自动创建。Chrome 安全策略不允许普通安装包静默安装插件，所以第一次需要手动按步骤加载。

常见问题

  1. 双击后没有网页：不要关闭黑色窗口，把 data\logs 目录截图或打包发给维护人员。
  2. Chrome 未自动打开：手动访问 http://127.0.0.1:8790/simple。
  3. 提示未登录：在自动打开的 Chrome 里扫码登录业务系统后，再回到操作页执行。
  4. 插件加载失败：请选择“解压后的文件夹”，不要直接选择 zip 文件。
  5. 更新失败 / 无网络：程序仍以当前版本启动，下次联网时会自动更新。
